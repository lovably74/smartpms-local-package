# SmartPMS LAN IP 접속

## 목적

같은 네트워크의 다른 PC나 모바일에서 개발 PC의 로컬 IP(예: `192.168.0.10`)로 브라우저 접속이 가능하도록 한다.

## 동작 요약

- HTTP 서버는 `0.0.0.0`에 바인딩되어 LAN에서 해당 포트로 들어오는 연결을 받는다.
- Vite 개발 서버는 `allowedHosts: true`로 설정되어 `Host` 헤더가 로컬 IP여도 차단되지 않는다.
- `localhost`만 쓰려면 환경 변수 `HOST=127.0.0.1`로 제한할 수 있다.

## 사용 방법

1. PC의 IPv4 주소 확인(Windows: `ipconfig`).
2. 개발 서버 실행 후 콘솔에 출력되는 `LAN (same subnet):` 아래 URL로 접속.
3. 방화벽에서 해당 TCP 포트(기본 3000) 인바운드 허용이 필요할 수 있다.

## 환경 변수

| 변수   | 설명                                      | 기본값    |
|--------|-------------------------------------------|-----------|
| `HOST` | 바인딩 주소. LAN 허용은 `0.0.0.0` 유지 권장 | `0.0.0.0` |
| `PORT` | 서버 포트                                 | `3000`    |
